# Brand images

Served by Home Assistant itself (2026.3+) via `/api/brands/integration/koolnova/<image>`;
these local images take priority over the brands CDN, so the integration shows its
logo without a PR to home-assistant/brands (which no longer accepts custom integrations).

Files (Koolnova's official logo, used for identification only):
- `icon.png` (256×256) / `icon@2x.png` (512×512) — KN monogram, square.
- `dark_icon.png` / `dark_icon@2x.png` — light version for dark backgrounds.
- `logo.png` / `logo@2x.png` — full wordmark (height 256 / 512).
- `dark_logo.png` / `dark_logo@2x.png` — light wordmark for dark backgrounds.

Source: https://koolnova.com/ (official brand assets).
